# Cullexa Picture Manager v1.0.0 — Release Notes

**Release Date:** June 4, 2026  
**Platform:** Windows 10/11 (64-bit)  
**Binary Name:** `Cullexa-Picture-Organizer-Setup-1.0.0.exe`  

---

## Overview
We are thrilled to announce the initial release of **Cullexa Picture Manager (v1.0.0)**. Cullexa is a lightweight, AI-powered desktop application designed to solve the most tedious part of a photographer's workflow: photo culling. 

Using localized machine learning models, Cullexa analyzes your image collections, scores each photo's visual quality, and provides comparative view modes to help you separate your keepers from the discards in minutes.

---

## Key Features in v1.0.0

### 1. AI-Powered Quality Scoring
* Automatically evaluates and scores every photo on a scale from **0 to 100**.
* Scoring is based on focus/sharpness, exposure, color composition, and facial expressions (preventing you from keeping closed-eye shots).
* Processing runs entirely locally on your machine—no cloud uploads, telemetry, or internet required for image analysis.

### 2. Multi-View Culling Experience
* **Grid View**: A high-level view showing all thumbnail images alongside their AI scores to quickly spot and filter out low-scoring photos.
* **Compare Mode**: Review two similar photos side-by-side with close-up sync-zooming to choose the sharper shot.
* **Slider View**: Standard full-screen view with a split-screen slider to compare before/after edits or adjacent shots.

### 3. Streamlined Culling Workflow
* **Quick Keyboard Shortcuts**: Mark images as *Accept* (keep), *Reject* (discard), or rate them using a 1-to-5 star rating system.
* **Smart Filter & Sort**: Sort by file size, date taken, filename, or AI quality score. Filter to display only high-scoring shots or un-flagged photos.
* **Safe Cleanup Recycle Workflow**: Cullexa marks rejected photos and moves them to a temporary deletion queue. Deleted files are safely sent to the Windows Recycle Bin rather than permanently erased.

---

## System Requirements
* **Operating System:** Windows 10 or Windows 11 (64-bit only).
* **Processor:** Intel Core i3 / AMD Ryzen 3 or higher.
* **Memory:** 4 GB RAM minimum (8 GB RAM recommended for large libraries).
* **Storage:** ~150 MB free disk space for installation.

---

## Installation Instructions
1. Download **`Cullexa-Picture-Organizer-Setup-1.0.0.exe`** from the GitHub Release page.
2. Double-click the installer to start setup.
3. Follow the on-screen prompts to choose installation path and create desktop shortcuts.
4. Launch **Cullexa Picture Manager** and select a folder of images to begin culling!
